# Initial Setup

This guide walks you through the initial configuration of camelAI after your first login.

## System Configuration

### 1. Organization Settings

Navigate to Settings > Organization and configure:

- Organization Name
- Time Zone
- Date Format
- Currency Format
- Language Preferences

### 2. Email Configuration

If you plan to use email notifications:

1. Go to Settings > Email
2. Configure SMTP settings:
   - SMTP Server
   - Port
   - Username
   - Password
   - TLS/SSL settings

### 3. Security Settings

Under Settings > Security:

- Enable/disable two-factor authentication
- Configure password policies
- Set up session timeout
- Configure IP restrictions

## Database Connections

### 1. Add Your First Database

1. Navigate to Databases > Add New
2. Select your database type:
   - PostgreSQL
   - MySQL
   - SQLite
   - Other supported databases

3. Enter connection details:
   - Host
   - Port
   - Database name
   - Username
   - Password

### 2. Test Connection

Before saving, test the connection to ensure:
- Credentials are correct
- Network access is available
- Required permissions are set

## User Management

### 1. Create Additional Users

1. Go to Users > Add New
2. Enter user details:
   - Username
   - Email
   - Role (Admin, Analyst, Viewer)
   - Password

### 2. Set Up Teams

1. Navigate to Teams > Add New
2. Create teams for different departments or projects
3. Assign users to teams
4. Set team-specific permissions

## Analysis Configuration

### 1. Configure Analysis Settings

Under Settings > Analysis:

- Set default analysis parameters
- Configure visualization preferences
- Set up data refresh intervals
- Define analysis templates

### 2. Set Up Data Models

1. Go to Models > Add New
2. Define your data models:
   - Table relationships
   - Field types
   - Calculated fields
   - Custom metrics

## Best Practices

### Security

1. Change default passwords
2. Enable two-factor authentication
3. Set up regular password rotation
4. Configure session timeouts

### Performance

1. Set appropriate cache settings
2. Configure data refresh intervals
3. Optimize database connections
4. Set up monitoring

### Backup

1. Configure automatic backups
2. Set up backup retention policies
3. Test backup restoration
4. Document backup procedures
